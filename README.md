# Moodle Learning Platform

## Project Description
This project is a modern educational website dedicated to the Moodle learning management system.

The website demonstrates the main Moodle features, online education opportunities, and platform advantages for students and teachers.

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- GitHub Pages

---

## Main Features

- Modern responsive design
- Navigation menu
- Hero section
- Feature cards
- Participants section
- Development journal
- Useful resources
- Footer section

---

## Website Pages

- Home
- About Moodle
- Features
- Participants
- Journal
- Resources

---

## Goal of the Project

The goal of this project is to create a professional educational website using basic web technologies and publish it using GitHub Pages.
